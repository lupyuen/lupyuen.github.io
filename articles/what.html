          Why ‚ÄúReal-Time‚Äù?

          It‚Äôs a Real-Time Operating System (RTOS) because NuttX was created for tiny microcontrollers: STM32, nRF52, BL602, ESP32, ‚Ä¶

          Which won‚Äôt run a General Purpose Operating System. (Like Linux)

          It‚Äôs probably more suitable for Education right now: For learning what happens when a Smartphone boots, how it renders graphics on the LCD Display.

          (Turning NuttX into Daily Driver might be an interesting project for students!)

          Does NuttX support X11? Wayland?

          Where did the code come from?

          We created the code based on the official docs for the Allwinner A64 SoC. But some parts of Allwinner A64 are poorly documented. (Like the Display Engine)

          A sizeable chunk of our code was Reverse Engineered from the p-boot Bootloader, by observing which Hardware Registers it uses.

          The NuttX Community is now adding new drivers for Allwinner A64 SoC, like for I2C.

          (We welcome your contribution to NuttX!)
